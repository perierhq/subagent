## Subagents

Delegate parallelizable, bulk, or review work to subagents via the `subagent` CLI instead of doing it inline. Before spawning one, load the `subagent` skill — it covers model selection (cost/intelligence/taste rankings) and mechanics (read-only mode, background runs, JSON output).
